> Boilerplate for Fullstack website development - NodeJS version

### 🏠 [Homepage](https://github.com/datproto/Boilerplate---Fullstack/tree/node)

## Prerequisites

- npm >=8.1.2
- yarn >=1.12.17
- node >=16.13.2

## Install

```sh
yarn
```

## Usage

```sh
yarn start:admin
yarn start:backend
yarn start:frontend
```
